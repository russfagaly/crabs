# Pitch Count Tracker — 2026 Alameda 12u All-Stars

## Little League 12u Pitch Count Rules

| Pitches in a day | Rest required |
|-----------------|---------------|
| 1–20            | No rest required |
| 21–35           | 1 calendar day |
| 36–50           | 2 calendar days |
| 51–65           | 3 calendar days |
| 66+             | 4 calendar days |

**Maximum per day:** 85 pitches (12u)

A calendar day of rest means the player may not pitch again until the required number of days have passed, regardless of how many games are played.

---

## How to Use

The pipeline automatically tracks each pitcher's pitch count from the `pitches` field in every game file. After running `compile.py`, the Player Pitching tab will show season pitch totals.

For tournament-specific day-by-day tracking, maintain this log manually:

---

## Key Notes

- A pitcher who is removed from the mound **cannot return as a pitcher** in the same game
- If a player starts catching after pitching, **rest day rules change** — check current LL rules
- Pitch counts must be confirmed with the official scorekeeper after each game
- Keep this log updated in real-time during tournament play — eligibility errors can result in forfeit
